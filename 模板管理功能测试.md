# 模板管理功能测试指南

## ✅ 已修复的问题

**问题**：模板管理中的编辑功能打不开
**原因**：操作按钮没有绑定点击事件处理函数
**解决方案**：为所有操作按钮添加了正确的onClick事件处理

## 🔧 修复内容

### 1. 编辑按钮
```typescript
// 修复前（无点击事件）
<Button type="text" icon={<EditOutlined />} />

// 修复后（添加点击事件）
<Button
  type="text"
  icon={<EditOutlined />}
  onClick={() => handleEdit(template)}
/>
```

### 2. 预览按钮
```typescript
<Button
  type="text"
  icon={<EyeOutlined />}
  onClick={() => handlePreview(template)}
/>
```

### 3. 复制按钮
```typescript
<Button
  type="text"
  icon={<CopyOutlined />}
  onClick={() => handleCopy(template)}
/>
```

### 4. 删除按钮
```typescript
<Button
  type="text"
  danger
  icon={<DeleteOutlined />}
  onClick={() => handleDelete(record)}
/>
```

## 🧪 测试步骤

### 访问应用
1. 打开浏览器访问：http://localhost:3002
2. 点击"模板管理"标签页

### 测试编辑功能
1. **点击编辑按钮**：
   - 在任意模板行找到编辑按钮（铅笔图标）
   - 点击该按钮
   - **预期结果**：弹出编辑对话框，显示模板的当前信息

2. **测试编辑功能**：
   - 修改模板名称或描述
   - 点击"确定"保存
   - **预期结果**：模板信息更新成功，表格中显示更新后的内容

### 测试其他功能
1. **预览功能**：
   - 点击预览按钮（眼睛图标）
   - **预期结果**：显示预览提示信息

2. **复制功能**：
   - 点击复制按钮（复制图标）
   - **预期结果**：创建模板副本，名称后缀"- 副本"

3. **删除功能**：
   - 点击自定义模板的删除按钮（垃圾桶图标）
   - **预期结果**：弹出确认对话框，确认后删除模板

4. **新建功能**：
   - 点击"新建模板"按钮
   - **预期结果**：弹出新建对话框

## 📋 测试数据

当前系统中有3个默认模板：
- **标准审批单**（默认模板，不可删除）
- **请假审批单**（自定义模板）
- **费用报销单**（自定义模板）

## 🎯 功能说明

### 编辑功能
- 可以编辑模板名称和描述
- 默认模板也可以编辑（但可以设为非默认）
- 编辑后自动更新时间戳

### 复制功能
- 复制模板会创建一个新的模板
- 名称自动添加"- 副本"后缀
- 新模板为自定义模板（非默认）

### 删除功能
- 只能删除自定义模板
- 默认模板不允许删除
- 删除前会弹出确认对话框

### 预览功能
- 当前显示简单的提示信息
- 后续可以扩展为真实的模板预览
